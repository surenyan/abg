## 2.2. Ոսկե հատվածի եղանակը

Այս եղանակի դեպքում հատվածը որոշ կետով բաժանվում է երկու անհավասար մասի, որոնց համար ամբողջ հատվածի երկարությայն և այդ մասերից մեծի երկարությայն հարաբերությունը հավասար է այդ երկու մասերից մեծի և փոքրի երկարությունների հարաբերությանը։

Ուկե հատույթի եղանակի դեպքում $[a, b]$ հատվածի վրա վերցվում են նրա ծայրակետերի նկատմամբ սիմետրիկորեն դասավորված $x_1$ և $x_2$ կետեր այնպես, որ տեղի ունենան հետևյալ առնչությունները՝

$$\frac{b-a}{b-x_1} = \frac{b-x_1}{x_1-a} = \frac{b-a}{x_2-a} = \frac{x_2-b}{b-x_2}$$

Ուստի դժվար չե համոզվել, որ $x_1$ կետը $[a, x_2]$ հատվածի ոսկե հատույթի երրորդ կետն է, իսկ $x_2$ կետը $[x_1, b]$ հատվածի ոսկե հատույթի առաջին կետը։ Այնհայտ է, որ առաջին և երրորդ կոտրակների հաշվաարությունից բխում է նաև

$$x_1 = a + b - x_2$$

կամ

$$x_2 = a + b - x_1$$

հաշվաարությունը։

Այժմ ենթադրենք, որ $f(x)$-ն ունիմոդալ է $[a, b]$ հատվածի վրա, և անհրաժեշտ է գտնել նրա min-ի $x^*$ կետը $\varepsilon > 0$ բացարձակ սխալանքի ճշտությամբ։

---

**_Քայլ 0_**: Ընդունենք $a_0 = a$, $b_0 = b$ և գտնենք ոսկե հատույթի $x_1^{(0)}$ և $x_2^{(0)}$ կետերը։ Ոսկե հատույթի առնչությունների առաջին երկու կոտրակի հաշվաարությունից կունենանք

$$x_1^{(0)} = \frac{3 - \sqrt{5}}{2} \cdot (b_0 - a_0) = a_0 + 0{,}3819 \cdot (b_0 - a_0)$$

իսկ վերջին երկու կոտրակի հաշվաարությունից՝

$$x_2^{(0)} = a_0 + \frac{\sqrt{5}-1}{2} \cdot (b_0 - a_0) = a_0 + 0{,}6180 \cdot (b_0 - a_0)$$

Ուստի կարող ենք հաշվել $f(x_1^{(0)})$ և $f(x_2^{(0)})$ արժեքները։

---

**_Քայլ 1_**: Որոշենք նոր $[a_1, b_1]$ հատվածը և $x_1^{(1)}$ ու $x_2^{(1)}$ կետերը, ինչպես նաև՝

- ա) եթե $f(x_1^{(0)}) \leq f(x_2^{(0)})$, ապա ընդունենք $a_1 = a_0$, $b_1 = x_2^{(0)}$, $x_2^{(1)} = x_1^{(0)}$, $x_1^{(1)} = a_1 + b_1 - x_1^{(0)}$
- բ) եթե $f(x_1^{(0)}) > f(x_2^{(0)})$, ապա $a_1 = x_1^{(0)}$, $b_1 = b_0$, $x_1^{(1)} = x_2^{(0)}$, $x_2^{(1)} = a_1 + b_1 - x_2^{(0)}$

---

**_Քայլ k ($k \geq 1$)_**: Այնուհետևն՝

- ա) եթե $f(x_1^{(k-1)}) \leq f(x_2^{(k-1)})$, ապա ընդունենք $a_k = a_{k-1}$, $b_k = x_2^{(k-1)}$, $x_2^{(k)} = x_1^{(k-1)}$, $x_1^{(k)} = a_k + b_k - x_1^{(k-1)}$
- բ) եթե $f(x_1^{(k-1)}) > f(x_2^{(k-1)})$, ապա $a_k = x_1^{(k-1)}$, $b_k = b_{k-1}$, $x_1^{(k)} = x_2^{(k-1)}$, $x_2^{(k)} = a_k + b_k - x_2^{(k-1)}$

---

Որոնումն ավարտվում է, երբ որոնման ընթացքի բայլերի քանակը $(n)$ բավարարում է հետևյալ պայմանին՝

$$|b_n - a_n| \leq \varepsilon$$

իսկ որպես min-ի $x^*$ կետի մոտավորություն ընդրվում է $x_m = 0.5 \cdot (a_n + b_n)$ կետը, որի դեպքում $f^* = f(x_m)$։
