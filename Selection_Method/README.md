Ընտրանքների եղանակը

Ենթադրենք `f(x)`-ն սահմանված ֆունկցիա է `[a, b]` հատվածի վրա, և անհրաժեշտ է գտնել նրա մինիմումը ε > 0 ճշտությամբ:

`[a, b]` հատվածը բաժանում ենք `n` հավասար մասերի.

$$x_k = a + k \cdot \frac{b-a}{n}, \quad k = \overline{0,n}, \quad n \geq \frac{b-a}{\varepsilon}$$

Հաշվում ենք `f(x)`-ի արժեքները այդ կետերում և գտնում ենք մինիմումը.

$$f(x_m) = \min_{k=\overline{0,n}} f(x_k)$$

Առավելագույն սխալը հաշվվում է հետևյալ բանաձևով.

$$\varepsilon_n = \frac{b-a}{n}$$

Ճշտության պայմանը.

$$|x_m - x^*| \leq \varepsilon$$

Եթե այս պայմանը բավարարվում է, ապա `x_m`-ը համարվում է ֆունկցիայի մինիմումի մոտավոր կետը։
